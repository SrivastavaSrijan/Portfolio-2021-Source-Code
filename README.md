## Getting started
1. Install [nvm](https://github.com/creationix/nvm#install-script).
2. Install npm and node.

    ```
    nvm install 8
    nvm use 8
    ```
3. Install yarn.

    ```
    npm install -g yarn
    ```

### Developing with the browser example

1. Start development
    ```
    cd <DIR>
    yarn start
    ```

Open http://localhost:3000 in the browser.
